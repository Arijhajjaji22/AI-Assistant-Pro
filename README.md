# AI Assistant Pro

**Plugin VS Code** qui utilise l'IA Gemini pour analyser, corriger et expliquer ton code comme un développeur senior.

---

## 🎯 Fonctionnalités

1. **Détection d'erreurs automatique**  
   À chaque sauvegarde (`Ctrl+S`), le plugin analyse ton code et souligne les erreurs.

2. **Correction automatique**  
   Clique sur l’ampoule 💡 pour corriger automatiquement les erreurs détectées.

3. **Explication du code**  
   Sélectionne un bloc de code et appuie sur `Ctrl+Shift+E` pour comprendre son fonctionnement.

4. **Refactoring intelligent**  
   Sélectionne du code “fonctionnel mais moche”, appuie sur `Ctrl+Shift+R`, et l’IA le réécrit proprement.

---

## 💻 Stack technique

- JavaScript / TypeScript  
- Node.js  
- VS Code API  
- API Gemini (IA)
